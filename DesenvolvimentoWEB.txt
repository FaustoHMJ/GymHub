Nome do projeto: Gym Hub

 

Descrição: O projeto busca entregar uma solução simples para academias de qualquer tipo. O projeto busca check-in para aulas, reserva de equipamentos, planos disponíveis, contrato atual e vitrine virtual para divulgação de produtos disponíveis nas academias que possuem uma loja.
